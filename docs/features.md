# CleanIndia Connect — Feature List

## Core Features for Version 1

### 1. Landing Page

A clean and attractive introduction page explaining the mission of CleanIndia Connect.

### 2. Report Garbage Issue

Users can submit a cleanliness issue by entering:

* Issue title
* Description
* City or area
* Issue type
* Urgency level
* Image proof

### 3. Image Preview

Users can upload an image and preview it before submitting the report.

### 4. Reports Feed

All submitted reports appear as cards with title, location, issue type, urgency, image, and status.

### 5. Report Status

Each report can have one of these statuses:

* Pending
* In Progress
* Resolved

### 6. AI Complaint Generator

Users can enter a simple issue description, and the platform generates a formal complaint format that can be copied and submitted to official portals.

### 7. Sustainability Tips

Users can view and share simple tips related to cleanliness, waste reduction, recycling, plastic-free living, and sustainable habits.

### 8. Community Discussion

Users can comment on reports or share suggestions to solve local cleanliness problems.

### 9. Dashboard

The dashboard shows:

* Total reports
* Pending reports
* Resolved reports
* Cleanliness impact score

### 10. Government Submission Help

The platform provides guidance and links to official complaint portals. In Version 1, the platform will help users prepare complaint text but will not automatically submit complaints to government systems.
